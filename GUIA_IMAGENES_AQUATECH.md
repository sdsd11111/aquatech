# 📐 Guía de Optimización de Imágenes — Aquatech

Para mantener la estética **Apple Premium** y asegurar que la web cargue instantáneamente, sigue estas especificaciones para las imágenes.

## 📁 Estructura de Archivos
- **Carpeta Local:** `tmp-assets/` (Pon las fotos aquí para que yo las suba).
- **Formato Obligatorio:** `.webp` (Máxima compresión sin pérdida de calidad).

---

## 🖼️ Especificaciones por Sección

### 1. Hero Slider (Carrusel Principal - Computadora)
- **Nombres:** `hero-slider-1.webp`, `hero-slider-2.webp`, `hero-slider-3.webp`
- **Dimensiones:** **2560 x 1440 px** (4K) o **1920 x 1080 px**.
- **Aspecto:** 16:9 (Horizontal).

### 2. Hero Slider (Celular - Vertical)
Opcional pero recomendado para que la foto no se corte en móviles.
- **Nombres:** `hero-mobile-1.webp`, `hero-mobile-2.webp`, `hero-mobile-3.webp`
- **Dimensiones:** **1080 x 1920 px** o **750 x 1334 px**.
- **Aspecto:** 9:16 (Vertical).

### 2. Producto Estrella (Fondo Negro)
- **Nombre:** `pro-product.webp`
- **Dimensiones:** **1600 x 1200 px**.
- **Composición:** Producto centrado con iluminación dramática. Si es posible, con fondo negro puro (`#000000`) para que se funda con la sección.

### 3. Bento Grid (Contenido 50/50)
- **Nombres:** 
    - `locales-lifestyle.webp`
    - `hidromasaje-card.webp`
    - `piletas-card.webp`
    - `accesorios-card.webp`
- **Dimensiones:** **1200 x 1000 px**.
- **Relación de Aspecto:** 6:5 o 1:1.

---

## 🚀 Tips de Optimización
1. **Calidad WebP:** Al exportar a WebP, usa una calidad de entre **75% y 85%**. Es el punto dulce entre peso y nitidez.
2. **Peso del Archivo:** 
    - Las fotos del Hero no deberían pesar más de **300-400 KB**.
    - Las fotos de las tarjetas no deberían pesar más de **150-200 KB**.
3. **Herramientas recomendadas:** [Squoosh.app](https://squoosh.app/) (de Google) o [TinyPNG](https://tinypng.com/).

---

**¡Avisame cuando las fotos estén en la carpeta `tmp-assets` para procesar la subida!**
